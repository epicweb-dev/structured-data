# Reduce

ğŸ‘¨â€ğŸ’¼ The `reduce` method accumulates an array into a single value. It's the most
powerful (and sometimes confusing) array method.

```ts
const numbers = [1, 2, 3, 4]
const sum = numbers.reduce((acc, n) => acc + n, 0)
// 10
```

Parameters:

- **accumulator** (`acc`) - The running total
- **current** (`n`) - The current element
- **initial value** (`0`) - Starting point
